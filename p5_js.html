"p5.js is a JavaScript library for creative coding... p5.js has a full set of drawing functionality" (p5js.org). Using p5.js, I created a self-portrait, a drawing canvas with unique brushes, and a basic game.



Birb Flap Game


I recreated a version of Flappy Bird, an old mobile game where the player's goal is to navigate a bird through some obstacles and obtain as high a score as possible. To me, Flappy Bird represents an internet fad, which is often recreated and mimicked until it's so prolific that people get tired of it and it becomes irrelevant.

How to Play

Press any key to start. Press any key to move the bird upwards, and navigate the bird between the gaps of each pillar to earn points.

DIY Photoshop


The inspiration for my DIY photoshop canvas was to enable the user to draw a simple anime-like face, with a simplified head shape, anime-style eyes, and a few classic expressions, along with a color picker, line tool, and erase tool. The user can also save the image they create.

Controls

Keys 1: Head Tool    Keys 2-4: Anime Eye Tool    Keys 5-6: Nose Tool    Keys 7-9: Mouth Tool    Key 0: Line Tool    Key e: Erase Tool    Key x: Clear Canvas    Key p: Save Image

Self Portrait


The idea behind the self-portrait was that the portrait could be used as an icon to represent myself online, both as an artist and as an individual. I felt that using a simple and cute design was in line with my own personality, with only a few small details to make the portrait feel more vibrant.
